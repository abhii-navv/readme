<br>

An intelligent web application that accepts wildlife photographs and performs deep image quality analysis using computer vision — then outputs actionable, data-driven editing recommendations tailored to professional tools like **Adobe Lightroom** and **Snapseed**.

<div align="center">

| Attribute | Details |
|:---|:---|
| **Stack** | Python, Flask, OpenCV, NumPy, HTML, CSS, JavaScript |
| **Domain** | AI-Powered Web Application · Computer Vision |

| **Interface** | Browser-based image upload with live analysis results |
| **Deployment** | Render (Live Production) |

The backend pipeline uses **OpenCV** to compute image quality metrics — sharpness, luminance, and tonal balance — then maps those metrics to specific slider values across multiple editing platforms. Designed for wildlife and nature photographers seeking a technical edge in post-processing.

<summary><b>⬡ Uber Ride-Sharing Simulation System</b> &nbsp;|&nbsp; Core Java · OOP · Data Structures · CLI</summary>

<br>

A terminal-based, object-oriented simulation of a ride-sharing platform modeled after real-world systems like Uber — replicating user registration, driver assignment, ride booking, parcel delivery, and fare estimation workflows entirely in Core Java.

<div align="center">

| Attribute | Details |
|:---|:---|
| **Stack** | Core Java, OOP, Data Structures, CLI |
| **Domain** | Systems Design · Object-Oriented Engineering |
| **Design Principles** | Encapsulation, Abstraction, Inheritance, Modular Classes |
| **Core Features** | Ride booking, driver availability, fare estimation, parcel delivery |
| **Interface** | Fully interactive CLI simulation engine |
| **Architecture** | Custom route mapping and service selection logic |

</div>

Designed and implemented a modular class hierarchy that mirrors real-world ride-sharing operations. Applied core **OOP principles** rigorously — encapsulation for user/driver data, abstraction for service interfaces, and inheritance for vehicle type hierarchies. Implemented fare computation using custom route-distance mapping and built a CLI engine that allows live simulation of the full booking lifecycle.
